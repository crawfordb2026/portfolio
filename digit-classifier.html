Neural Network Digit Classifier

Real-time handwritten digit recognition with deep learning

Live Demo GitHub

Overview

Built an end-to-end machine learning web application that accurately predicts handwritten digits (0-9) using a custom neural network trained on the MNIST dataset. Users can draw digits on an interactive canvas and receive instant predictions with confidence scores, achieving ~95% accuracy.

Key Features

Interactive Drawing

HTML Canvas for seamless digit drawing with real-time predictions

Confidence Visualization

Dynamic bar charts showing prediction confidence for each digit

Mobile-Responsive

Touch-friendly interface optimized for mobile devices

Production Ready

Live web application deployed and accessible via browser

Technologies Used

Backend

Python Flask NumPy

Frontend

HTML CSS JavaScript Canvas API

Machine Learning

PyTorch Neural Networks MNIST Dataset

Results & Impact

Successfully deployed a working AI application accessible via web browser

Demonstrates full-stack ML development capabilities

Showcases ability to transform research models into user-facing products

Real-time performance with instant predictions and ~95% accuracy
