Stock Price Predictor

Machine Learning for Stock Market Forecasting

Live Demo

Overview

A comprehensive machine learning project that predicts stock prices using time series analysis and technical indicators. The system fetches historical stock data, engineers meaningful features, and trains multiple ML models to forecast next-day closing prices with 18+ technical indicators including RSI, Bollinger Bands, and Moving Averages.

Key Features

Data Processing

Automated stock data collection and preprocessing with robust error handling

Feature Engineering

Creates 18+ technical indicators including RSI, Bollinger Bands, Moving Averages, and Volume Trends

Machine Learning

Implements and compares Linear Regression and Random Forest algorithms with comprehensive evaluation

Real-time Prediction

Generates next-day price forecasts with confidence intervals and interactive visualizations
